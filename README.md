# EcoSnap
Users can upload images of items they want to recycle or donate.


AI-Powered Identification — The application uses an AI tool to identify the item in the image and suggests the appropriate recycling or donation options based on its properties and local guidelines.


Location-Based Suggestions — Suggests nearby recycling centers, donation centers, or charities based on the user's current location and the type of item.


Information Display — Shows details such as opening hours, contact information and accepted items for each recycling or donation location.


User Profiles — Allows users to create profiles to track their recycling and donation activities.


Gamified Rewards — Implement a point system where users earn points for each successful recycling or donation.
